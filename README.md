- ### 🖥 Rodar o Jogo
    - 01-Clone

            git clone https://github.com/wwwillian/Jogo_C_Forca.git

    - 02-Entre na pasta

            cd Jogo_C_Forca

    - 03-Rode o comando para compilar o arquivo em C

            gcc main.c mapa.c ui.c -o main.out 

    - 04-Rode o jogo

            ./main.out
